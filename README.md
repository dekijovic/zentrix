TO DO Aplikacija, uradjena u Node.js sa MongoDB.

sve sto treba da bi se podigla aplikacija je pokrenuti docker i port 3000 da bude otvoren

``docker compose up -d``

U aplikaciji je dodat EJS kao template Engine za Views

Web stranica je:

`http://localhost:3000`
